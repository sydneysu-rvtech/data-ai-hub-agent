Getting Started for New Developers

Environment Setup

# Clone repository
git clone <repository-url>
cd data-ai-hub-agent

# Install dependencies
poetry install

# Set up development configuration
cp conf/dev.yml.example conf/dev.yml
# Edit conf/dev.yml with your settings

Local Development

# Run data pipelines
python -m data_ai_hub_agent.main --pipeline bronze
python -m data_ai_hub_agent.main --pipeline silver
python -m data_ai_hub_agent.main --pipeline gold

# Run tests
poetry run pytest

# Run the Slack bot locally
databricks bundle run --target dev
