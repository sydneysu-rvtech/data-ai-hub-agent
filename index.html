1. Knowledge Ingestion via Scheduled Databricks Data Pipeline Jobs

Slack

Databricks

Official docs & KB articles

2. User Asks a Question

Developer Tags @Data-AI-Hub Bot

The posted question is sent to the server and triggers the RAG workflow.

3. Query Knowledge Base Using Vector Search

Semantic Search & Retrieval

Converts question to embeddings and finds most relevant knowledge chunks.

4. RAG Processing & AI Generation

Prompt + Retrieved Knowledge → AI Engine

Combines user question with relevant context to generate accurate, informed responses.

5. Slack Response Delivery

Real-Time 24/7 Integration

Delivers instant, contextual answers via FastAPI & WebSocket connection.
